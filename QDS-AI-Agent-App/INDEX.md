## ğŸ’¡ Pro Tips

1. **First Time Building?**
   - Allow 10-15 minutes for first build
   - First build downloads dependencies
   - Subsequent builds are faster

2. **Testing on Device**
   - USB device faster than emulator
   - Enable USB Debugging in Settings
   - Real device more reliable for testing

3. **Backend Development**
   - Test /api/health with cURL first
   - Use BACKEND_INTEGRATION.md for examples
   - Start with simple health endpoint

4. **Customization**
   - Change app name in strings.xml
   - Customize colors in colors.xml
   - Modify default IP in MainActivity.kt

5. **Troubleshooting**
   - Run `gradlew clean` if issues occur
   - Delete `.gradle` folder if stuck
   - Check Android Studio version compatibility
